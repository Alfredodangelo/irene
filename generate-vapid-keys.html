Genera VAPID Keys

Le VAPID keys servono per le notifiche push. Genera una coppia di chiavi, poi copia la Public Key nel file push-config.js e la Private Key nelle variabili di n8n.

Come usare le chiavi

  1. Clicca "Genera nuove chiavi" qui sopra
  2. Copia la Public Key e incollala nel file js/push-config.js al posto di INSERISCI_QUI_LA_VAPID_PUBLIC_KEY
  3. Copia la Private Key e salvala come variabile d'ambiente in n8n: VAPID_PRIVATE_KEY
  4. In n8n, aggiungi anche queste variabili:
    VAPID_PUBLIC_KEY = la stessa Public Key
    VAPID_SUBJECT = mailto:irenegipsytattoo@gmail.com
  5. Crea la tabella push_subscriptions in Supabase (vedi SQL nel file push-config.js)
